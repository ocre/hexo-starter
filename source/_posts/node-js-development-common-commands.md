#### node多版本管理工具 nvm
* nvm list
* nvm current
* nvm install stable
* nvm install 9.2.0
* nvm use 9.2.0

#### 包管理工具 npm
* npm list -g --depth 0
* npm install webpack -g
* npm install # 安装依赖
* npm run init  # 项目初始化
* npm run dev  # 运行测试环境
* npm run build  # 生产环境打包
